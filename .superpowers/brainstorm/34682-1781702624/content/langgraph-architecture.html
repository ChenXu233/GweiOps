🤖 LangGraph 自驱动 Agent 架构

Agent 自主决策，不是固定流程

核心理念

不是：先做 A → 再做 B → 再做 C（固定流程）

而是：定义目标 + 工具 + 状态，Agent 自主决定路径

🎯 关键设计点

✅ Agent 自主决策

  • 根据 Issue 内容判断类型
  • 选择是否需要复现
  • 选择复现策略
  • 生成三种 Patch 方案
  • 遇到问题自主处理

⚠️ 人类决策点

  • 选择 Patch 方案（三种之一）
  • 确认 PR 内容
  • 处理异常情况
